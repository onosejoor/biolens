# Bio-Lens Website

A modern landing page for Bio-Lens featuring a sophisticated design with royal blue, gold, and black color scheme.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```
